InfoAmazonia API Documentation

Comprehensive API documentation for the InfoAmazonia Admin Dashboard with AI-powered search capabilities.

GET /api/article-stats
Retrieve comprehensive statistics about articles in the database including total count, oldest and newest publication dates.

Response Model

{ "success": boolean, "stats": { "total_count": integer, "oldest_date": "string (DD/MM/YYYY)", "newest_date": "string (DD/MM/YYYY)" }, "error": "string | null" }

Example Response

{ "success": true, "stats": { "total_count": 1250, "oldest_date": "15/01/2024", "newest_date": "10/06/2025" }, "error": null }
POST /api/search
Search articles using full-text search with optional AI-powered summary generation. Supports Portuguese and English queries.

Request Body

query
string (required)
Search term or phrase to find articles
generate_summary
boolean (optional, default: false)
Whether to generate an AI-powered summary of search results
system_prompt
string (optional)
Custom system prompt for summary generation

Example Request

{ "query": "mudanças climáticas", "generate_summary": true, "system_prompt": "Provide a detailed analysis" }

Response Model

{ "success": boolean, "results": [ { "id": "string", "title": "string", "similarity": number (0.0-1.0), "url": "string", "short_url": "string", "published_date": "string (YYYY-MM-DD)", "author": "string", "description": "string", "key_words": ["string"] } ], "count": integer, "summary": "string | null", "error": "string | null" }

Example Response

{ "success": true, "results": [ { "id": "123e4567-e89b-12d3-a456-426614174000", "title": "Impactos das Mudanças Climáticas na Amazônia", "similarity": 0.85, "url": "https://example.com/article", "short_url": "/admin/articles/123e4567-e89b-12d3-a456-426614174000", "published_date": "2024-06-15", "author": "João Silva", "description": "Análise detalhada dos efeitos...", "key_words": ["clima", "amazônia", "meio ambiente"] } ], "count": 1, "summary": "Encontrado 1 artigo sobre mudanças climáticas...", "error": null }
GET /search-articles
Render the search articles page with authentication. Returns HTML template for the search interface.

Authentication

Cookie
access_token
JWT token for admin authentication

Response

Content-Type: text/html Status: 200 OK HTML template with search interface
